# detect-indent-cli

> Detect the indentation of code


## Install

```
$ npm install --global detect-indent-cli
```


## Usage

```
$ detect-indent --help

  Usage
    $ detect-indent <file>
    echo <string> | detect-indent

  Example
    $ echo '  foo\n  bar' | detect-indent | wc --chars
    2
```
